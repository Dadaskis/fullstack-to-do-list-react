# Fullstack To-Do List Application

A complete task management application built with React, featuring a full set of CRUD operations and a clean, responsive user interface.

## Features

- Create new tasks with titles and descriptions
- View all tasks in an organized list
- Edit existing tasks
- Delete tasks with confirmation
- Modal-based forms for all operations
- Responsive design for mobile and desktop use

## React Components

- **App.jsx** - Main application component that orchestrates the entire to-do list functionality
- **TaskManager.jsx** - Core component managing the state and logic for all task operations
- **TasksList.jsx** - Displays all tasks in a structured list format
- **Task.jsx** - Individual task component showing task details and action buttons
- **AddTaskForm.jsx** - Form component for creating new tasks
- **EditTaskForm.jsx** - Form component for modifying existing tasks
- **RemoveTaskForm.jsx** - Confirmation form for task deletion
- **Modal.jsx** - Reusable modal component for displaying forms
- **utilities.jsx** - Helper functions and utility methods
- **server.jsx** - Mock server simulation for API interactions

## Technologies Used

- React for building the user interface
- CSS modules for component-specific styling
- Vite as the build tool and development server
- ESLint for code quality and consistency

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
4. Open your browser to the localhost address shown in the terminal

## Project Structure

The application follows a component-based architecture with clear separation of concerns. Each component has its own CSS file for styling, and utility functions are centralized in a separate file for reusability.
